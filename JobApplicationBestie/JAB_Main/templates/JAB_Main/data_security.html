Data Security

1. Introduction

We take data security seriously and only use industry standards technologies to protect your data. This page provides an overview of our data security practices and the steps we take to safeguard your information.

2. Secure Data Transmission

All data transmitted between your device and our servers is encrypted using secure protocols (e.g., HTTPS) to prevent unauthorized access during transmission.

3. Data Storage

We store your data on secure servers with restricted access. Access to your data is limited to authorized personnel only.

4. User Authentication

We use strong authentication mechanisms to ensure that only authorized users can access your data. Passwords are hashed and stored securely, and we encourage users to use strong and unique passwords.

5. Reporting Security Issues

If you discover any security vulnerabilities or have concerns about the security of your data, please report them to us immediately at: rohan.rathore93@gmail.com
